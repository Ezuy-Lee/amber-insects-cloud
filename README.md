#### 简介
 
为什么叫昆虫？之前看过《昆虫记》一书，觉得昆虫特别有趣，本人特别喜欢昆虫里面的甲虫。当然昆虫也是世界上数量最多、种类最多、形态最多的节肢动物，已发现100多万种，它们的踪迹几乎遍布世界的每一个角落。
